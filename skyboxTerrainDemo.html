Unity: Textures

Textures should be sized in powers of 2. For reasons of technical efficiency, graphics chips like to handle textures in sizes that are 2N: 4, 8, 16, 32, 64, 128, 256, 512, 1024, 2048... In your image editor (Photoshop, GIMP...). You can use non-square Textures like 512 x 64, just as long as the size remains a power of 2. Popular resource for textures - textures.com and TextureHaven

Image Resolution, recommended texture formats

Mobile PC
Max 2048
(common for both mobile and PC)		 2048, 4096, 8192, 16384
HDR - high definition render packages commonly uses 4096
recommendations according to graphics processing units (GPS)
My recommendation is to stick with 2048 unless you are targeting high-end computers, then go with 4096. You can always downsize, but you can not upgrade bitmaps without loss of quality

Textures can be part of materials, and materials are applied to geometry. A material is a set of information defining the properties of a surface; that information can include a texture to display on that surface. Typically each texture goes with a different material, so for convenience Unity allows you to drop a texture onto a game object in the scene and then it creates a new material automatically.

Texture coordinates are an extra set of values for each vertex that assign polygons to areas of the texture image. Think about it like wrapping paper; the 3D model is the box being wrapped, the texture is the wrapping paper, and the texture coordinates represent where on the wrapping paper each side of the box will go. From Unity in Action: texture coordinates in appendix C.

Unity: Textures

Albedo: technical term for the base color, all the shadows and highlights have been removed
Normal Map: use RGB channels to create the illusion of depth. The dark to light values in each the Red, Green and Blue channels are used to have lighter areas pop out whereas dark values recede to create a visual representation of depth. Generate using Photoshop: Filter > 3D > Generate Normal Map - violet - in Unity must be imported as Texture Type: Normal map.
Height or Bump Map: use luminosity to create the illusion of depth. Bump maps read light and dark areas - luminous values - white creates the illusion of pushing out the pixels - pushing them at you, black the illusion of pushing in the pixels away from you. Although it is a 2D image, light and dark are used to simulate depth or as if it was a 3D object - as if it had depth on the z-index. Generate using Photoshop: Filter > 3D > Generate Bump (Height) map - grayscale - white areas representing the high areas of your texture and black representing the low areas.
(The default settings for Blur (0.5) and Detail (80%) tend to work well, but you should experiment with different values and apply them in Unity to achieve the level of detail that you wish.)

What are the different texture maps for?; Secondary Maps and Detail Mask

When adjusting tiling on a game object, make sure you’re setting the tiling of the main map and not the secondary map (this material supports a secondary texture map for advanced effects). The reason for secondary maps is to allow the material to have sharp detail when viewed up close, while also having a normal level of detail when viewed from further away, without having to use a single extremely high texture map to achieve both goals.

Unity: Skybox, Unity in Action Chapter 4

New Material, Shader > Skybox > 6 Sided

A skybox is a cube surrounding the camera with pictures of the sky on each side. No matter what direction the camera is facing, it’s looking at a picture of the sky.

Select imported texture to see its properties in the Inspector, and change the Wrap Mode setting from Repeat to Clamp. Be sure to click Apply. Ordinarily, textures can be tiled repeatedly over a surface; for this to appear seamless, opposite edges of the image bleed together. But this blending of edges can create faint lines in the sky where images meet, so the Clamp setting will limit the boundaries of the texture and get rid of this blending.

Apply the new Skybox: Top Unity Menu - Window > Rendering > Lighting > Environment tab and select your new Skybox Material to apply it to the scene.

Toggle on or off in skybox, fog and effects button at top of Scene view.

Alternatively, you could create a Procedural Sky - this is entrely based on Material settings: in the Assets folder create a new material and in its Inspector select Shader Skybox > Procedural and play with the settings - Sun, Sun Size, Atmosphere, Sky Tint, Ground, Exposure. Apply it by going to Window > Rendering > Lighting > Environment tab and select your new Procedural Skybox as the Skybox Maerial

Unity: Create Natural Environment by Terrain Sculpting

  1. In your Scene Hierarchy, Create Terrain (default 1000 meters)
  2. For demonstration, I will change the Terrain settings to a Terrain Width, Length and Height of 100.
  3. With Terrain selected, in the inspector click the padlock - the lock button - this way you can focus on creating your terrain and not worry about accidentally deselecting the terrain while you are sculpting it.
  4. Inspector > Raise or Lower Terrain - uniform vs noisy brushes
  5. Modify the Stroke:
    • Brush Strength - hold down A (how fast the brush works)
    • Brush Size - hold down S
    • Brush Spacing
    • Brush Rotation - hold down D
  6. Steps to think about when working on a Terrain:
    • Start with larger brush and larger area of Terrain
    • Mountains vs flat area vs water
    • Modify size and rotation as you go
    • Change Brush Mask as you go to diversify the landscape
  7. The Terrain game object offers many brushes to work with, a few examples are:
    • Brushes > Raise and Lower Terrain
    • Brushes > Paint Texture and Create Terrain Layers to create a diverse topography
    • Set Height - sets the height of the terrain, say for example you want to add a foundation for a structure or settlement, the Set Height will level the selected area - flatten it
  8. Create a neighboring terrain by selecting the Neighboring Terrain button. Click in the scene view to create an adjacent terrain. This expands your terrain to create a more massive world with the adjacent terrains fluidly welded to your initial terrain.

Unity: Painting Terrain Textures

  • Select the Paint Brush Tab in the Terrain Component
  • In the Inspector, scroll down to > Add Layer - defaults from asset package - dirt, moss, rock, sand, scree (broken rock fragments at the base of mountain cliffs...), snow
  • Repeat to add each layer
  • Paint the textures onto the Terrain
  • Start with foundation layers such as rock and dirt and work up to the surface layers such as sand and snow.
  • Do so by selecting the various layers and their materials to build out environment. Try different brush masks and stroke settings so that things aren't uniform.
  • Modifying tilling, duplicate a layer, such as rock, bring up the tilling size and overlay with the new layer
