Unity for VR

XR Interaction Toolkit

VR and AR development has changed quickly over the past two years. Two years basic interaction required a good deal more work and decision making than it currently does. As Janet Murray emphasizes throughout Hamlet on the Holodeck as technology advances and the programmers and engineers build tools that facilitate the creation of new computer-based experiences, mediamakers and artists are without extensive programming skills can create new experiences and forms of storytelling. I wish that Unity's XR Interaction Toolkit had existed two years ago as countless hours of programming and wiring would have been saved...

An ideal way to jump in is Setting up a VR-Enabled Project with Unity and the XR Interaction Toolkit that uses a prepared project: Unity-Technologies XR-Interaction-Toolkit-Examples

Unity for VR

XR Interaction Toolkit

  • Install from Package Manager > Unity Registry:
    XR Interaction Toolkit
    Oculus XR Plugin
    Note on developing for Oculus Quest - you must download the Oculus app to your phone and pair it with your headset and in the phone app go to Settings and turn on Developer tools; this will allow you to build directly from Unity to your Oculus Quest when it is tethered via USB
  • From the top menu go to Edit > Project Settings, click on XR Plugin Management, click on Android tab and select Oculus
  • Open the Build Settings, target Android as Oculus is an Android device
  • Click on Player Settings and in Other Settings set the minimum level to Android 6.0 'Marshmallow' (API level 23)
  • In Build Settings, if Oculus is connected select it in Run Device and click Build and Run

This project will get quickly get you up and running. However a tutorial that walks you step by step in creating varous forms of interaction is VR Beginner: The Escape Room (this is a Universal Render Pipeline project), using this 61MB package

There are of course third party VR Interaction Frameworks that may offer more ready-made scripts and interactivity. One with high ratings and many reviews is VR Interaction Framework
